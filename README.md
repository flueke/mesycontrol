# mesycontrol - Remote control for mesytec NIM devices.

See https://mesytec.com/products/mesycontrol/mesycontrol.html for more info and downloads.

## Dependencies

* System packages:

  - libboost-dev
  - libprotobuf-dev protobuf-compiler
  - libqwt-qt5-dev
  - libxcb-util
  - patchelf

* Python dependencies

  - PySide2
  - cx-Freeze
  - pyqtgraph
  - protobuf
